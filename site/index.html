Why Defacto?

Designed for systems programming with full control over hardware

Bare-Metal Development

Write operating systems and bootloaders directly in Defacto. Full control over x86-32 hardware without runtime overhead.

Automatic Memory Management

The compiler automatically frees all variables at the end of each section. No manual memory management or garbage collection required.

Struct Types

Define custom data structures with named fields and dot notation access. Perfect for system data representation.

Hardware Drivers

Built-in support for keyboard, mouse, and volume drivers with simple, declarative syntax.

Multiple Modes

Compile for bare-metal kernel, Linux terminal, or macOS terminal with syscall support. One language, multiple targets.

Native Addons

Extend Defacto with libraries written in Rust or C++ for maximum performance and ecosystem integration.

Clean, readable syntax for low-level programming

Defacto 
#Mainprogramm.start
#NO_RUNTIME
#SAFE

<.de
    var msg: string = "Hello, World!"
    static.pl>
    display{msg}
.>

#Mainprogramm.end
#Mainprogramm.start
#NO_RUNTIME
#SAFE

struct Player {
    x: i32
    y: i32
    health: i32
    score: i32
}

<.de
    var player: Player
    static.pl>
    player.x = 100
    player.y = 200
    player.health = 100
    player.score = (player.score + 100)
.>

#Mainprogramm.end
#Mainprogramm.start
#NO_RUNTIME
#SAFE

<.de
    var x: i32 = 5
    var msg: string = "equal"
    var err: string = "not equal"
    static.pl>

    if x == 5 {
        display{msg}
    } else {
        display{err}
    }
.>

#Mainprogramm.end
#Mainprogramm.start
#NO_RUNTIME
#SAFE

<drv.
Const.driver = keyboard_driver
keyboard_driver <<func = keyboard>>
.dr>

<.de
    var key: i32 = 0
    static.pl>
    call #keyboard_driver
    readkey{key}
    display{key}
.>

#Mainprogramm.end

Quick Install

Get up and running in minutes

Windows

Download and run the installer

Download .exe

Requires NASM: choco install nasm

macOS

Install via Homebrew

xcode-select --install
brew install nasm mingw-w64
cd compiler && make
./defacto -h

Linux

Ubuntu/Debian

sudo apt update
sudo apt install build-essential nasm
cd compiler && make
./defacto -h
