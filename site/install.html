Defacto v0.30 — Installation

Defacto is a low-level programming language for x86-32, bare-metal experiments, and custom toolchains. Choose your platform below to get started.

Current Version v0.30 pre-alpha
Release Date 2026
License Open Source

Download & Installation

Windows

Download Installer (.exe) v0.30

Requirements

  • NASM assembler (install via: choco install nasm)
  • Windows 10 or later
  1. Download and run the installer
  2. Follow the installation wizard
  3. Add Defacto to your PATH
  4. Verify: defacto -h

macOS

Install via Terminal v0.30

Requirements

  • Xcode Command Line Tools
  • Homebrew (recommended)
xcode-select --install
brew install nasm mingw-w64
git clone https://github.com/vivooifo-droid/Defacto.git
cd Defacto/compiler
make
./defacto -h

Linux

Build from Source v0.30

Requirements

  • Build essentials (gcc, make)
  • NASM assembler
  • Git
# Ubuntu/Debian
sudo apt install build-essential nasm git

# Fedora/RHEL
sudo dnf install gcc make nasm git

# Arch Linux
sudo pacman -S base-devel nasm git

# Then build
git clone https://github.com/vivooifo-droid/Defacto.git
cd Defacto/compiler
make
./defacto -h

Quick Start

Once installed, create your first Defacto program:

// hello.de
#Mainprogramm.start
#NO_RUNTIME
#SAFE

<.de
    var msg: string = "Hello, World!"
    static.pl>
    display{msg}
.>

#Mainprogramm.end

Compile and run:

./defacto hello.de
./a.out
