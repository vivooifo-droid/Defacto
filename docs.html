Установка

Первый шаг к работе с языком — установка инструментов разработки. В этом руководстве мы рассмотрим все доступные способы установки для различных операционных систем.

Системные требования

Перед установкой убедитесь, что ваша система соответствует минимальным требованиям:

  • Операционная система: Windows 10+, macOS 10.15+, Linux (любой современный дистрибутив)
  • Оперативная память: минимум 2 ГБ (рекомендуется 4 ГБ)
  • Свободное место: минимум 5 ГБ на диске
  • Компилятор C: требуется для некоторых пакетов

Установка на macOS

Самый простой способ установки на macOS — использование менеджера пакетов Homebrew:

Терминал 
brew install rust-style

Или используйте официальный установщик:

Терминал 
curl --proto '=https' --tlsv1.2 -sSf https://rust-style.org/install.sh | sh

Установка на Linux

Для большинства дистрибутивов Linux рекомендуется использовать официальный скрипт установки:

Терминал 
curl --proto '=https' --tlsv1.2 -sSf https://rust-style.org/install.sh | sh

Для Ubuntu/Debian также доступен пакет в репозитории:

Терминал 
sudo apt install rust-style

Установка на Windows

Скачайте установщик для вашей архитектуры:

Скачать для Windows x64 Скачать для Windows ARM64

Или используйте winget:

PowerShell 
winget install RustStyle.RustStyle

Проверка установки

После установки проверьте, что всё работает корректно:

Терминал 
rust-style --version

Вы должны увидеть версию компилятора: 

rust-style 1.93.1 (abc1234 2024-01-01)
