Defacto Language Documentation

Welcome to the official documentation for the Defacto programming language. Defacto is a low-level programming language designed for x86-32 architecture, bare-metal development, and custom toolchains.

Pre-Alpha Status: Defacto v0.30 is under active development. The language and toolchain are unstable and will change.

Quick Start

Here's a minimal Defacto program:

#Mainprogramm.start #NO_RUNTIME #SAFE <.de var msg: string = "Hello, World!" static.pl> display{msg} .> #Mainprogramm.end
