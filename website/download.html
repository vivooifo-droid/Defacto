macOS

Version 0.30.0

Install Dependencies

xcode-select --install
brew install nasm make

Build from Source

cd compiler
make
./defacto -h
Alternative: Download the source from GitHub and build manually.

Linux

Version 0.30.0

Ubuntu/Debian

sudo apt update
sudo apt install build-essential nasm make

Build from Source

cd compiler
make
./defacto -h
Other distros: Install gcc, make, and nasm using your package manager.

One-Line Installation

macOS / Linux

# One-line install (direct from GitHub) bash -c "$(curl -fsSL https://raw.githubusercontent.com/vivooifo-droid/Defacto/main/defacto-installer.sh)" # Or download and run manually curl -LO https://raw.githubusercontent.com/vivooifo-droid/Defacto/main/defacto-installer.sh bash defacto-installer.sh

Verify Installation

# Check installation defacto -h # Expected output: Defacto compiler help

Quick Start

Compile Your First Program

# Create hello.de #Mainprogramm.start #NO_RUNTIME #SAFE <.de var msg: string = "Hello, World!" static.pl> display{msg} .> #Mainprogramm.end # Compile for terminal (Linux) defacto -terminal hello.de # Run ./hello
